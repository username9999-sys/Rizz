## 🚀 HOW TO USE NEW FEATURES

### AutoML Example
```python
POST /api/automl/run
{
  "dataset_id": "data123",
  "task_type": "classification",
  "time_limit": 3600,
  "ml_frameworks": ["sklearn", "xgboost"]
}
```

### DEX Swap Example
```javascript
POST /api/dex/swap
{
  "poolId": "pool123",
  "tokenIn": "0x...",
  "amountIn": 100,
  "user": "user123",
  "slippage": 0.5
}
```

### Live Stream Example
```bash
POST /api/live
{
  "title": "My Stream",
  "category": "gaming",
  "recordStream": true
}
```
